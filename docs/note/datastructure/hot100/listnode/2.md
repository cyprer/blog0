## 问题:k个一组翻转链表
    给你链表的头节点 head ，每 k 个节点一组进行翻转，请你返回修改后的链表。

    k 是一个正整数，它的值小于或等于链表的长度。如果节点总数不是 k 的整数倍，那么请将最后剩余的节点保持原有顺序。

    你不能只是单纯的改变节点内部的值，而是需要实际进行节点交换。

我的代码:
```java
class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        ListNode dummy = new ListNode(0, head);
        ListNode pitr = head;
        int n = 0;
        while (pitr != null) {
            n++;
            pitr = pitr.next;
        }
        ListNode gPrev = dummy;
        ListNode gHead = head;
        for (; n >= k; n -= k) {
            ListNode prev = null;
            ListNode curr = gHead;
            for (int i = 0; i < k; i++) {
                ListNode next = curr.next;
                curr.next = prev;
                prev = curr;
                curr = next;
            }
            gPrev.next = prev;
            gHead.next = curr;
            gPrev = prev;
            gHead = curr;
        }
        return dummy.next;
    }
}
```
> 问题在于对gPrev赋值为prev了,这里实际上是一组的末尾了,对于下一组的gPrev应该gHead(翻转过来的话).

> 最好在循环内定义gHead嗯;
---
> 另一种写法(头插法),相当于遍历链表将其插在prev的后面.
```java
class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        ListNode dummy = new ListNode(0, head);
        ListNode prev = dummy;
        int n = 0;
        for (ListNode p = head; p != null; p = p.next) n++;
        for (; n >= k; n -= k) {
            ListNode cur = prev.next;
            for (int i = 1; i < k; i++) {
                ListNode nxt = cur.next;
                cur.next = nxt.next;
                nxt.next = prev.next;
                prev.next = nxt;
            }
            prev = cur;                                                        
        }
        return dummy.next;
    }
}
```

# 第二周周结
  回家的第一周,玩的很嗨皮,学习上很迷茫,不知道该干点啥...
## 1.15
  周四，天气晴，中午起来收拾行李准备回家，回去直接找亮去了，等了一会阿必个曦也回来了，阿必先鸿坤小区找洗个澡，浩浩他们家也搬去鸿坤啦，我们就顺便去看看，唠嗑一会就去亮四舅家吃炒粉，真的香，好久没吃了，吃完又去福民夜市逛，到处逛，到处逛，然后准备找个地方打王者，兜兜转转只能去个奶茶店打王者，一直赢，输了就回去亮家睡觉，但是没那么睡着，一直在聊天，阿必一直很遗憾没加上火车上遇见的那个女孩，就一直聊，本来都聊到六点准备吃饭了，他们两个直接睡着了，我也是没招了，然后我也睡了
## 1.16
  中午起来就去东中吃个快餐，回来继续睡觉，睡觉起来就去虾塘帮忙做饭，嗯我就先回去洗澡再来了，吃完就回亮家躺尸，然后晚上我和亮和必和曦和家俊去糖人街吃清补凉，无话可说，只能打王者，感觉氛围很差，也不知道为啥，然后家俊先回去了，我们四个就先去网吧玩，玩两个小时就回来了。然后就聊天，聊了很多，比昨天要多，爱情友谊国家都有，只能说这样的时光真是屈指可数了吧，虽然很困，但也很珍惜，聊到早上甚至，直接吃个早餐才回来睡觉了
## 1.17
  周六，天气晴，起来已经两点多了，去吃点小吃就回来躺尸了,下午就去虾塘那里吃饭,晚上打打王者,本来要早点睡了,亮又跟我说要不要去吃夜宵,我刚好也嘴馋就一起去了,路上顺便去东昌买了点烧烤吃,这才是海南生活,回来刷抖音很晚才睡着的.
## 1.18
  周日,天气晴,起来吃的方便面,吃完继续睡,睡醒就让亮拉我回去了,回来收拾一下,嗯打了会王者,然后吃饭洗澡,写日记,接着打王者.
## 1.19
  周一,天气晴,昨天睡得挺晚的,但是10点就醒了,刷了会抖音才起来的,接下来就看那个go的ai项目吧,稍微看了一下,然后吃饭睡觉了,三点醒了,去看一下爷爷奶奶,爷爷现在都无法起身走路了,害,然后和奶奶聊会天,虽然她讲的人和事我有很多都不懂,但是我还是陪陪她吧,嗯配了一个多小时我也就回去了,然后家俊叫我打了会王者,打完我就去学做菜,吃完饭刷了会抖音,然后继续看项目,然后晚上出去吃夜宵,半路没电了,没招啊,然后去亮家充电,我和亮和黑和曦就去糖品街吃夜宵,我点了个鸡屎藤汤圆,我大概是第一次吃鸡屎藤吧,就是用面粉做的,像鸡屎一样,味道一般吧,避雷了,然后打了会牌,差不多就回去了,车不够回家了就在黑家睡了一晚
## 1.20
  周二,天气晴,九点就醒了,然后刷会抖音回家了,接下来还是继续学习java技术吧,不整什么ai了,先找到实习再说吧,先学点kafka吧,诶,看了和没看一样...我真不到该学什么了...好迷茫...算了再去看看elasticsearch吧,学不下了打了把王者然后吃饭,接着写会力扣,三点醒了,去学一个langchain4j的医疗项目,没看一会老黑叫我打王者,没办法,打吧,真学不动了,打完做会饭,然后吃饭,刷会抖音,分享一个视频给亮,说有个人长得像陈婕,没想到他在那悦,然后让我过去,然后我去了,发现他在摘豆角,我就去帮他,嗯忙活了一会,他妈妈给了我一点玉米豆角还有青菜给我拿回去,然后回来洗个澡,继续学习了,看了会老黑叫我去吃汤圆,我就去了,吃完回来打王者.
## 1.21
  周三,天气晴,昨晚听小说睡着了,虽然有点人机声,然后中午才起来,手机都听没电了,接着吃完饭写写力扣,然后睡觉了,昨晚睡得有点晚,以至于中午睡得有点昏沉,有点赖床,睡了很久,下来也没学习,打起王者了,接着继续学习做饭,嗯继续打王者,然后洗澡,看起小说了,看的盘龙.

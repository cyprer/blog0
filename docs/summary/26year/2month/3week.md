# 第三周周结
  过年周?爽玩了.
## 2.15
  周日,天气晴,中午醒的,吃个饭,一直玩觉醒之战,然后入迷看了JOJO,一直在看.
## 2.16
  周一,天气晴，直接被炮轰醒了,晚会大巴扎,下午还在看JOJO,但看着看着又不想看了,下来玩了会月圆之夜,然后就吃饭了,然后洗个澡,接着就写力扣了,虽然今天是除夕,但也要好好学习哈哈哈,太吵了学不了一点,下去玩了,晚上去老黑家打麻将.
## 2.17
  周二,天气晴,中午吃完饭去少仕家打麻将,晚上去打亮家打炸胡,难得阿某出来打,不过十一点就回去了,然后亮叫卢炳晶来吃夜宵,总感觉这人很熟悉呢,吃完还下雨了,在两家睡觉,聊天很多都是现实问题了,终究还是要面对阿.
## 2.18
  周三,天气晴,10点起床回家了,然后吃完饭去浩浩家打牌,还以为说要在名山吃饭了,结果妈妈说她没来,那我也就回去那悦吃饭了,然后洗个澡,就去八所打麻将,和卢炳晶打,不过打到十一点就说回去了,然后我和亮和顺就三个人打,打到一点就去吃夜宵,顺请客说下午去名山赌赢1k块哈哈哈,然后就回家睡觉了.
## 2.19
  周四,天气晴,早上十点就被炮轰醒了,然后继续写力扣,吃完午饭顺叫我去接他去虾塘坐着,坐了一下午,亮他妈妈给我一个50的红包,在虾塘吃完饭就去看比赛,然后说帮忙记分可以给100块的红包,爽歪歪了,然后也没什么活动,就回去了,开的是亮的车,然后我洗完澡就去八所了.
## 2.20
  周五,天气晴,十点多醒了,然后黑来接我,今天要去必业家拜年,陆陆续续接了浩和曦就出发了,到了就开始打牌.我去了,按一根一块算输了两百,还好说超过一百就是五毛,不然输臭了,然后不敢打了,然后亮和顺和丹虹也来了,他们就开始玩三公,我觉得不好玩就不玩了,也不敢玩,晚上就开造,虽然我没咋吃,但是后面也是喝了好多,四点才睡的.
## 2.21
  周六,天气晴,11点多醒的,也没吃饭,我和顺和必就一起去不磨了,到了吃点饭,然后他们来就开始打牌,我就不玩了,玩不了一点,很快就吃饭了,4点多就煮好了,大概是昨天喝怕了,然后没啥人喝酒,吃完我就睡着了,好困,醒了看到他们还在打牌,晚上黑就接我回去,然后睡觉.

# 第三周周结
  这周依旧是算法和项目呢,但真的很幸运,之前面的一家小厂说要我了,我也就去了,很兴奋,但还是很焦虑,每晚都焦虑得睡不着.
## 3.15
  周日,天气晴,中午才醒,玩了会大巴扎睡了,三点做pdd笔试,做了一会不想做了,做牛魔,真的很烦这个什么输入输出,四道编程题,就算做出来也进不去,我做个毛,不如玩会大巴扎,大巴扎有bug,有的物品不显示,难受,不玩了,但是还是很想玩,然后不断尝试,卸载重装终于恢复了,然后今晚就一直在玩大巴扎,也没锻炼,摆完了.
## 3.16
  周一,天气晴,中午醒的,嗯先写力扣吧.写了一会本来要睡觉了,家俊跟我吐槽他舍友打瓦不睡午觉说要换宿舍就叫我打王者,然后打了一个小时我也不睡了,下来看知光项目了,写写力扣,然后弄一下暑期实习的投递记录,晚上还是看八股,下课跑步锻炼洗澡,继续投递,不投了,先写力扣吧,写滑动窗口最大值呢,写着写着发现自己不会用双端队列了,不知道是不是被豆包教坏了,自己写了没毛病,豆包好蠢.
## 3.17
  周二,天气晴,虽然又失眠了,但是定了九点的闹钟起来看八股,舍友醒了就写力扣,很快就睡觉了,下午把大营销看了,回顾发现xfg讲的是真一般,接着再把简历修改一下,然后接着写算法,然后尝试去运行知光项目,docker是弄好了,不过还没启动就去上课了,上课还是看八股,看的真的乏了,打了会王者,第三节课亮打视频给我叫我打王者,于是我逃课了,打了一晚上,也没学习了.
## 3.18
  周三,天气晴,早上九点就醒了,但是舍友还在睡,但是我又不想看八股,于是我就去看xfg的组件项目视频,看完不知咋的玩起大巴扎了,一玩就玩到睡觉,我直接不去上课了,让越峰帮我签到,于是下午四点才开始学习,先写力扣,写了好久,几把算法咋这难,感觉效率好慢,诶,然后跑步锻炼去了，亮又打视频叫我打王者,于是我就打王者,洗澡,然后开始投简历.
## 3.19

## 3.20
